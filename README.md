# Simple Calculator Web App

This is a simple and responsive calculator web application built using HTML, CSS, and JavaScript.  
It performs basic arithmetic operations with a clean and user-friendly interface.

## Features
- Addition, Subtraction, Multiplication, Division
- Clear (AC) and delete functionality
- Responsive design for mobile and desktop
- Interactive button clicks

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone the repository
2. Open `index.html` in your browser

## Project Structure
- index.html
- style.css
- script.js
